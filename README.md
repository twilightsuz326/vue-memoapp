## System
* Vue.js 2.6.12

## Memo
* サーバが Node 16 環境のため fibers は入れない。(Laravel 10が稼働しているため)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### 機能一覧
* メモ一覧表示
  * 階層構造に対応
* メモ登録
* メモ編集
* メモ削除
